This directory contains screenshots of the CSV Import Application.

## Screenshot Guidelines

- **Resolution:** 1920x1080 or higher for desktop, 375x667 for mobile
- **Format:** PNG or JPG
- **Naming:** Use kebab-case (e.g., `upload-interface.png`)
- **Content:** Show the most important features and modern UI elements

## How to Take Screenshots

### Windows
- Use `Windows + Shift + S` for Snipping Tool
- Or use `Windows + Print Screen` for full screen

### Mac
- Use `Command + Shift + 4` for area selection
- Use `Command + Shift + 3` for full screen

### Browser DevTools
- Open DevTools (F12)
- Use device toolbar to capture mobile views
- Use responsive design mode for different screen sizes
